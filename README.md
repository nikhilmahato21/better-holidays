# Tripee Holidays – Travel Website

A modern, responsive travel agency website built with **React + Vite**, **Tailwind CSS**, and **Framer Motion**.

## Tech Stack
- ⚛️ React 18 + Vite 5
- 🎨 Tailwind CSS v3
- 🎬 Framer Motion v11
- 🔤 Google Fonts (Playfair Display + Outfit)

## Features
- 🖼️ Auto-sliding Hero with 4 destination backgrounds
- 📦 8 Tour Packages with real pricing (grid layout)
- 🗺️ Featured Destinations bento grid
- ⏱️ Live countdown timer Flash Deals section
- 💼 About + Why Choose Us with stats
- 📬 Contact form + footer
- 💬 WhatsApp floating button
- 🎯 Smooth scroll animations via Framer Motion (`useInView`)
- 📱 Fully responsive (mobile-first)

## Setup

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Build for production
npm run build
```
